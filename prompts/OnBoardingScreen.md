**Objective: Create Onboarding Screen Flow**

1. After the splash screen finishes, the application must automatically redirect the user to the onboarding screen.

2. The onboarding flow must consist of **at least four (4) steps/screens**, each containing **placeholder text** for content.

3. The first onboarding screen must display a primary action button labeled **“Get Started”**, while the succeeding screens must use navigation buttons such as **“Next”**, **“Back”**, or other appropriate labels.

4. Each onboarding screen must include the following elements:

   * **Header (Title)**
   * **Paragraph (Description)**
   * **Action Button**

5. All content elements (header, paragraph, and button) must be positioned at the **bottom-center of the screen**, ensuring consistent alignment across all onboarding steps.

6. The layout should maintain a **clean, minimal, and user-friendly design**, with proper spacing and readability.

7. Ensure smooth transitions between onboarding steps to provide a seamless user experience.
