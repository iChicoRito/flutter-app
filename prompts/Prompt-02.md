## Objective: Whole Project – Button Consistency

### 1. Button Inspection Across All Pages

Inspect the **Dashboard**, **Tasks**, **Spaces**, and **Profile** pages.  
Go inside each page and check **all buttons** present on those pages.

### 2. Enforce Button Consistency

Make all buttons consistent in the following aspects:

- **Colors** – Ensure uniform color usage across all buttons (e.g., primary, secondary, destructive, etc.)
- **Size** – Ensure consistent sizing across the whole project.  
  Specifically, maintain consistent dimensions for:
  - **Large** buttons
  - **Medium** buttons
  - **Small** buttons
