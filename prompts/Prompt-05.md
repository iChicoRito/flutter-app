# Objective
## Top Navbar Dashboard Dynamic Greetings

---

## Description
Provide a structured approach to making the greeting in the top navbar of the Dashboard/Home Page dynamic. The goal is to update the greeting message based on the time of day so that it reflects appropriate variations and enhances the user’s immersive experience.

---

## Objectives Breakdown

### 1. Main Objective Area
Make the greeting text in the top navbar of the Dashboard/Home Page dynamic based on the time of day.

---

### 2. Secondary Objective Area
Ensure the greeting message updates according to defined time ranges and displays the correct message alongside the user’s name.

---

### 3. Supporting Tasks
Break down the objective into actionable tasks based ONLY on what is stated.

#### 3.1 Task Group
- Identify the current greeting in the top navbar of the Dashboard/Home Page  
- Modify the greeting to be dynamic instead of static  
- Apply time-based conditions for greeting variations  
- Ensure the greeting displays with the user’s name  

---

### 4. Detailed Breakdown
Provide a deeper breakdown for more complex parts of the objective.

#### 4.1 Time-Based Greeting Logic
Define greeting variations based on time ranges:
- 1AM – 11AM: “Good Morning”  
- 1PM – 5PM: “Good Afternoon”  
- 6PM – 11PM: “Good Evening”  

#### 4.2 Navbar Integration
Ensure the dynamic greeting is displayed in the top navbar of the Dashboard/Home Page.

#### 4.3 User Experience Impact
Ensure the greeting contributes to a more immersive user experience.

##### Nested Details
- Maintain correct greeting output based on defined time ranges  
- Ensure greeting is consistently displayed with the user’s name  
- No additional behaviors beyond the stated requirement
