# Objective

## Additional Features and Re-layout

---

## Description

This objective focuses on enhancing the rich text editor by adding "Read Mode" and "Edit Mode" within the context menu, updating the "View Details" interface into a drawer or sheet based on a predefined design, and unifying the page layout by applying a consistent navbar header design across specified pages.

---

## Objectives Breakdown

### 1. Main Objective Area

Introduce additional functionality to the rich text editor and improve the overall layout and design consistency across related pages.

---

### 2. Secondary Objective Area

Ensure proper behavior for read and edit modes, update the presentation of detailed views, and align the user interface with an existing design for consistency.

---

### 3. Supporting Tasks

#### 3.1 Task Group

* Add "Read Mode" and "Edit Mode" options to the rich text editor context menu
* Implement behavior for "Read Mode" where editing is disabled and only reading, scrolling, and link interaction are allowed
* Implement behavior for "Edit Mode" where full editing capabilities are enabled
* Convert the "View Details" section into a drawer or sheet
* Follow the exact design specified in Figma for the drawer/sheet
* Apply the navbar header from "My Archives", "Edit Tasks", and "Add Tasks" to unify layout and design consistency across pages

---

### 4. Detailed Breakdown

#### 4.1 Read Mode Behavior

When "Read Mode" is selected, editing and typing are disabled for the owner. The user can scroll, read content, and interact with links only.

#### 4.2 Edit Mode Behavior

When "Edit Mode" is selected, the user is allowed to edit the content and perform all actions within the rich text editor.

#### 4.3 View Details Layout Update

The "View Details" interface is transformed into a drawer or sheet, following the exact design created in Figma.

##### Nested Details

* The drawer/sheet must strictly follow the provided Figma design
* No additional layout or design changes beyond what is specified
* The same page must use the navbar header from "My Archives", "Edit Tasks", and "Add Tasks" for consistent layout
