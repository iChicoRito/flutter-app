**Objective: Create Custom Splash Screen for Mobile App**

1. Upon launching the mobile application, a custom splash screen must be displayed immediately as the first screen the user sees.

2. The splash screen design must follow the provided color palette and apply the **60-30-10 color rule** correctly:

   * **60%** — `#1E88E5`
   * **30%** — `#90CAF9`
   * **10%** — `#E3F2FD`

3. The splash screen must include a **smooth, simple, and visually pleasing animation** that feels clean and professional.

4. The animation and splash screen display duration must last for **at least 5 seconds** before proceeding to the next screen.

5. The overall appearance should be modern, balanced, and consistent with the app’s branding, while keeping the design minimal and not overly crowded.

6. Ensure the splash screen transitions smoothly so the user experience feels polished and seamless when entering the app.
